Tecnologias que utilizamos

Conheça as tecnologias que utilizamos para desenvolver aplicações IoT.

Ethereum Blockchain

Ethereum é uma plataforma de código aberto, descentralizada e de alta performance. Nós utilizamos essa plataforma para prover a segurança das nossas aplicações IoT, e para aumentar a velocidade de nossas aplicações.

Arduino

Arduino é um microcontrolador de 8 bits, desenvolvido pela empresa Arduino. Ele é um dos microcontroladores que nós usamos em nossos protótipos e testes.

AWS - Amazon Web Services

O Amazon Web Services é uma plataforma de serviços de cloud computing, que permite que você crie e gerencie aplicações e serviços de cloud. Com a AWS é possível ter alta disponibilidade e baixa latência.
